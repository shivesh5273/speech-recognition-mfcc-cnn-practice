import numpy as np
from scipy.fft import fft, fftfreq
import matplotlib.pyplot as plt
In [3]:
# Constants
SAMPLE_RATE = 1000  # sampling rate (points per unit time)
DURATION = 10  # duration in time units
In [4]:
# Function to generate Gaussian function
def gaussian(x, sigma):
    return (1 / np.sqrt(2 * np.pi)) * np.exp(-x**2 / (2 * sigma**2))
In [5]:
# Generating time values
t = np.linspace(-DURATION/2, DURATION/2, SAMPLE_RATE * DURATION, endpoint=False)
In [6]:
# Calculating Gaussian for different sigma values
sigma_values = [1, 2, 4]
gaussians = [gaussian(t, sigma) for sigma in sigma_values]
In [7]:
# FFT computation
ffts = [fft(g) for g in gaussians]
frequencies = fftfreq(SAMPLE_RATE * DURATION, 1 / SAMPLE_RATE)
In [8]:
# Plotting the Gaussian functions
plt.figure(figsize=(12, 5))
for i, sigma in enumerate(sigma_values):
    plt.plot(t, gaussians[i], label=f'σ = {sigma}')
plt.title("Gaussian Functions for Different σ Values")
plt.xlabel("Time")
plt.ylabel("Amplitude")
plt.legend()
plt.grid()
plt.show()

# Plotting the absolute values of their Fourier transforms
plt.figure(figsize=(12, 5))
for i, sigma in enumerate(sigma_values):
    plt.plot(frequencies, np.abs(ffts[i]), label=f'σ = {sigma}')
plt.title("Absolute Values of Fourier Transforms of Gaussian Functions")
plt.xlim([0, 50])  # Limiting frequency range for better visibility
plt.xlabel("Frequency")
plt.ylabel("Magnitude")
plt.legend()
plt.grid()
plt.show()

The first plot shows the Gaussian functions for different values of σ (σ = 1, 2, and 4). As σ increases, the peak of the Gaussian function becomes lower and wider, indicating a more spread-out distribution in the time domain.¶

The second plot illustrates the absolute values of the Fourier Transforms (FFT) of these Gaussian functions. We observe that as σ increases, the corresponding Fourier Transform becomes narrower. This behavior is an example of the uncertainty principle in signal processing, where a function that is more localized in time (small σ) will have a wider spread in frequency, and vice versa.¶

For σ = 1, the FFT has a wider spread in frequency, indicating that the original Gaussian is more localized in time. As we increase σ to 2 and then 4, the original Gaussian function becomes less localized in time (more spread out), leading to a more localized (narrower) Fourier Transform in the frequency domain.¶
